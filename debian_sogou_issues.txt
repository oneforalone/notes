Debian + Sogou + error
1.安装fcitx
sudo apt install fcitx

2.安装im-config
sudo apt install im-config

3.运行im-config，选择fcitx输入法
im-config -c

4.安装搜狗输入法
去搜狗输入法官网下载deb包64位或32位，根据系统选择，点击安装。或
sudo dpkg -i deb包文件名

5.问题解决
如果不能输入中文或候选框不出现中文，可能是缺少libQtDeclarative.so文件。
sudo apt install libqt4-declarative

如果缺少依赖库：
sudo apt -f install
其他情况可搜索网络。
